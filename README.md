## 更换源
* 备份源列表 
``` shell
$ sudo cp /etc/apt/sources.list /etc/apt/sources.list_backup
```

* 更换源，编辑 `/etc/apt/sources.list`，用 `./sources.list` 中的替换(163源)
* 更新
``` shell
$ sudo apt-get update
```
