## 更换源
* 备份源列表 
``` shell
$ sudo cp /etc/apt/sources.list /etc/apt/sources.list_backup
```

* 更换源，用 `./sources.list` 中的替换
``` shell
$ sudo vim /etc/apt/sources.list
$ sudo apt-get update
```
